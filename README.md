# classutils
一个为高二 15 班专门开发的工具。

## 目前功能
- 随机抽取学生
 - 支持任意正确的数值
 - 支持指定性别
 - 支持设置是否允许重复

- 一键排座位
 - 以坐标形式输出学生座位
 - 支持导出为 xlsx 文件

## 计划功能
- 学生信息
- 优化 xlsx 输出格式
- 根据当天日期自动计算值日生
- 量化管理系统
- 连接 Ilya 的工地自动下载相关资源（如课表、值日表等）

## 依赖项
